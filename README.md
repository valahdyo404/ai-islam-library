## Getting Started

### Prerequisites
- Node.js >= 16.0.0
- npm

### Installation
1. Clone the repository
```bash
git clone [repository-url]
cd ai-islam-project
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
